This is a case-control outbreak investigation, so the correct measure of association is the odds ratio, not the risk ratio.

In a case-control study, the crude odds ratio is:

OR = (a d) / (b c)

where:

- a = exposed cases
- b = unexposed cases
- c = exposed controls
- d = unexposed controls
