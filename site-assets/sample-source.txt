Missing values

Some biological variables can contain zero or missing values that distort calculations.
For example, BMI = 0, BloodPressure = 0, or Glucose = 0 may need checking.

Useful commands:

is.na(x)
sum(is.na(x))
which(is.na(x))
mean(x, na.rm = TRUE)
na.omit(dataframe)
