What GPT-MD is

GPT-MD is a native macOS app that helps users select, rearrange, edit, preview, and export ChatGPT responses as Markdown. GPT-MD is designed for local desktop workflows involving macOS, Google Chrome, ChatGPT opened in Chrome, macOS Automation permission, optional managed Chrome mode, and MarkEdit for the intended export workflow.

Requirements

  • Use a compatible version of macOS.
  • Install Google Chrome and sign in to ChatGPT in Chrome.
  • Open the target ChatGPT tab before using GPT-MD.
  • Allow GPT-MD macOS Automation permission to control/read Google Chrome.
  • Enable Chrome's Allow JavaScript from Apple Events setting for the normal Chrome extraction path.
  • Install MarkEdit for the intended export workflow.

Permissions and managed Chrome

GPT-MD uses macOS Automation and browser automation mechanisms to inspect supported ChatGPT pages opened in Google Chrome. GPT-MD may also offer an optional managed Chrome mode using a separate Chrome profile and remote debugging. Managed Chrome mode does not remove the need to sign in to ChatGPT in that Chrome session.
